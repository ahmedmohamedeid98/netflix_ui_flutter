# netflix_ui_flutter

design the netflix home page with flutter and run it on mobile and web
=======

## important concept
* responsive ui
* bloc provider
* fade app bar opacity based on scrolling
* save state
* bottom navigation bar
* play, pause, mute, and unmute video
